# 繰り返し(for文)～沢山のものを動かす（２）～

## 基本形

```cs
 for (<初期化式> ; <繰り返し条件> ; <更新処理>) {
 //繰り返しブロック
 }
 
 //10回繰り返す具体例
 for (int i=0 ; i<10 ; i++){
    //iが0~9に変化しながら、ここを10回繰り返す
    }
 ```

## 処理順番
1. 初期化式を実行
2. 繰り返し条件が不成立ならfor文終わり
3. 繰り返すブロック実行
4. 更新処理を実行
5. 2へ戻る
